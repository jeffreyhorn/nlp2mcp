nlp2mcp API Documentation
==========================

Welcome to nlp2mcp's API documentation!

**nlp2mcp** is a Python tool that transforms Nonlinear Programming (NLP) models written in GAMS
into equivalent Mixed Complementarity Problems (MCP) by generating the Karush-Kuhn-Tucker (KKT) conditions.
